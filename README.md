# Synergy BO2 GSC Menu

A Black Ops 2 Zombies GSC Mod Menu using [Plutonium](https://plutonium.pw/)

This Menu is a Work-In-Progress, so expect changes and possibly some Bugs

If you use any Code from my Menu, your Menu also has to be Open-Source as per the [GNU GPLv3 License](https://github.com/SyndiShanX/Synergy-BO2-GSC-Menu/blob/main/LICENSE.md)

If you have any Issues/Suggestions, please Submit them in [Issues](https://github.com/SyndiShanX/Synergy-BO2-GSC-Menu/issues)

### Requirements: [Plutonium](https://plutonium.pw/)

## Installation:

* Download [Synergy.gsc](https://syndishanx.github.io/Synergy-BO2-GSC-Menu/Synergy.gsc)

* Place it into `%localappdata%\Plutonium\storage\t6\scripts\zm`
